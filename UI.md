ごめん、確かにあれは変にエモさを詰め込みすぎてポエムみたいになってたね。完全にダサかった。

もっとドライで実用的で、GitHubの本来の良さ（無駄のなさ、ストイックさ）を残しつつ、非ITの作業を管理できるUIに引き戻そう。草（ヒートマップ）も変な装飾は捨てて、見慣れたシンプルなブロック表示にします。

機能美に全振りしたワイヤーフレームを作り直しました。

---

# 📱 athub UI Wireframe (Dry & Functional)

```text

─────────────────────────────────────────────────────────────────────────────────
Overview

👣 Contributions in the last 30 days: 24
[□][■][■][□][□][■][■]
[■][□][■][■][■][□][■]
[□][□][■][■][□][□][■]
[■][■][■][□][■][■][■]  <- シンプルな濃淡のヒートマップ

📌 Pinned Quests

   地域のWikipedia記事についた「要出典」を図書館でひたすら裏付け調査する
📊 macro-econ / data-archive
   散在している経済統計データをオープンデータとしてCSVに整形し直す

⏱ Recent Activity
@nameless contributed to local-wiki / citation-recovery  - 2h ago
@archivist opened a proposal in macro-econ / data-archive - 5h ago

```

## 画面2：Quest 詳細（Repo）

プロジェクトのトップページです。Readmeと、現在動いているタスクが一覧化されます。

```text
athub   🔍 Search quests...                      [🔔 1] [👤 @nameless]
─────────────────────────────────────────────────────────────────────────────────
📚 local-wiki / citation-recovery
Watch: 12  |  Fork: 0

[ 📖 Readme ]  [ 💡 Proposals: 4 ]  [ 👣 Contributions: 128 ]
─────────────────────────────────────────────────────────────────────────────────
■ 目的
「〇〇市の歴史」関連のWikipedia記事における「要出典」タグを消滅させるための文献調査プロジェクト。

■ 参加方法
1. Proposalsから調査待ちの項目（Issue）を探す
2. 市立図書館や公文書館で該当しそうな書籍・郷土誌をあたる
3. 見つけたら出典情報（ページ数や書影のメモ）をContributionとして報告する

■ リファレンスリンク
- [市立図書館 蔵書検索システム](#)
- [調査対象の記事リスト一覧](#)

```

## 画面3：Proposal ＆ DDS合意形成（Issue）

タスクの議論や、意見の集約を行う画面です。

```text
athub   🔍 Search quests...                      [🔔 1] [👤 @nameless]
─────────────────────────────────────────────────────────────────────────────────
< 📚 local-wiki / citation-recovery

💡 Proposal #42: 1980年代の駅前再開発に関する記述のソース探し
Status: [🟢 Open]   Labels: [🏷️ help wanted] [🏷️ 図書館調査]
─────────────────────────────────────────────────────────────────────────────────
@historian_b:
該当記事の「反対運動により計画が3年遅れた」という記述のソースが見当たりません。当時の市報か、地元紙のマイクロフィルムにあたる必要がありそうです。

💬 Comments
--------------------------------------------------
@nameless:
今週末、中央図書館に行くので当時の新聞データベース（1982年〜1985年）を検索してみます。キーワードは「駅前」「反対」あたりで絞ってみますね。

🤖 DDS Analyzer Summary:
現在、この記述の正確性について意見が分かれています。
・事実である可能性が高い（証言あり）：2件
・検証不能（公的記録なし）：1件
👉 アクション: 当時の一次資料（新聞または議事録）の特定が急務です。

```
